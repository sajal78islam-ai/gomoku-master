# 🎮 Gomoku Master

Gomoku Master is a modern web-based strategy board game inspired by the classic five-in-a-row game.

## ✨ Features
- 🤖 AI opponent
- 🌙 Dark mode
- 📱 Mobile responsive
- 🏆 Leaderboard
- ✨ Smooth animations
## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Vercel
